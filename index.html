Welcome to MagicLand

After 25 years, massmind.org, a life long repository of learning by James Newton, has lost it's host and primary server. Sadface. Over time, I plan to move most of the valuable content to this github.com hosted site instead. For now, it's just the stuff I was already hosting on github.

  • The Hybrid Disk Encoder project An ultra low cost (laser cut) optical encoder easily supporting over 1M CPR ( less than 0.001 degree / 2 arcseconds ) with (optional) absolute position support. It does NOT require an FPGA.
  • Arduino Dynamixel / Stepper / IO Controller
  • The SDMG-Web-Bot An ESP based "battle" robot controller that uses your Cellphone webbrowers NOT an app.
  • WebBot A simple demonstration of a frame difference based vision system for the
    BlinkBot A stunningly cheap robot based on a smartphone using light blocks on a web page to control a cheap Arduino. See also: Related circuits / simulations.
  • TapCalc You need to drill a hole to tap for a bolt. But you don't have the exact drill the tap chart calls for. How close would the next drill size be? Or is there another drill that is ok, but using a different size chart? Or, what size should I drill for a really tight fit on the bolt? Or a loose fit? Or you have (gasp) non-standard bolt and need to find the hole for that. This page does that for you; you tell it the thread type, e.g. `#8-32` or `12mm` or whatever and it lists all the drill sizes of any standard that might work and how tight the fit will be:
  • ResistorCalc You need a specific resistance in a circuit. The standard values won't work, so you will need a pot or you can do a parallel set of resistors, but which two do you have that will get you at least near the target value? This page takes in a value and finds all the possible close matches.
